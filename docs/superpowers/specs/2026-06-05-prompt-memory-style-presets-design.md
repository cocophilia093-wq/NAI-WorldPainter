# 提示词辅助学习记忆与画风收藏设计

日期：2026-06-05

## 目标

本次改动扩展提示词辅助助手，使它能：

1. 在配置页点击“切换到此配置”时，同步把该配置设为提示词编排模型。
2. 记录用户纠正的人名、人物特征、画风等知识，下次命中相关词时优先使用本地记忆。
3. 移除聊天抽屉顶部“提示词辅助写手”标题和会话小标题，改为更紧凑的工具栏。
4. 增加画风画师串收藏页，可从相册添加压缩图片、保存标题和画风画师串，并一键添加到对话栏。
5. 历史画廊为空时显示“请去创造更多美好回忆吧！”，并提供“去创作”按钮。

## 方案

采用 A+C 结合方案：以本地轻量存储为底座，同时为学习记忆保留分类、搜索、覆盖和手动管理能力。

不引入远程服务，不依赖云端记忆。所有学习记忆和画风收藏都保存在本地。

## 学习记忆系统

### 数据模型

新增 `PromptMemory`：

- `id`：本地唯一 ID。
- `trigger`：触发词，例如角色名、别名、容易出错的人物特征。
- `content`：正确说明，作为下次生成时的高优先级依据。
- `type`：记忆类型，包含人名、人物特征、画风、其他。
- `source`：来源，包含手动、用户指令、LLM 候选。
- `createdAt` / `updatedAt`：时间戳。

### 记录入口

学习入口有三种：

1. 手动编辑：在提示词辅助工具栏进入记忆管理页，可新增、编辑、删除和搜索记忆。
2. 显式指令：用户在对话里说“记住：xxx”“纠正：xxx 应该是 yyy”时，系统解析并保存。
3. LLM 自判：对话中如果模型判断用户纠正了人名或人物特征，可生成记忆候选。候选需要通过可编辑入口确认后再长期保存，避免误记。

### 使用方式

用户发送纯文本或含图文本时，先用输入内容匹配本地记忆的 `trigger`。命中后，把匹配的记忆注入提示词辅助流程的系统上下文。

注入规则：

- 告知 LLM：若 Danbooru、模型常识或检索结果与本地记忆冲突，以本地记忆为准。
- 只注入命中的记忆，避免把全部记忆塞入上下文。
- 同一 `trigger` 存在多条时，优先使用 `updatedAt` 最新的手动编辑记录。

### 冲突处理

同 trigger 的新记忆覆盖旧记忆。手动编辑优先级最高，其次是用户显式指令，最后是 LLM 候选。

## 画风画师串收藏

### 数据模型

新增 `StylePreset`：

- `id`：本地唯一 ID。
- `title`：收藏标题。
- `prompt`：画风/画师串文本。
- `imagePath`：压缩后保存到 app 本地目录的图片路径。
- `createdAt` / `updatedAt`：时间戳。

### 添加流程

收藏页右上角提供新增入口：

1. 从系统相册选择图片。
2. 压缩图片并复制到 app 本地目录。
3. 用户填写标题和画风画师串。
4. 保存为 `StylePreset`。

### 页面与交互

在提示词辅助聊天抽屉顶部新增画风收藏入口。点击后进入新页面。

收藏页采用类似历史画廊的两列块状卡片：

- 卡片展示压缩图片和标题。
- 点击卡片进入详情页。
- 详情页支持放大查看图片、查看完整画风画师串、编辑、删除。
- 详情页提供“一键添加至对话栏”。点击后返回聊天抽屉，并把画风串插入输入栏，用户可继续编辑再发送。

## 聊天抽屉 UI

现有顶部标题区移除：

- 删除“提示词辅助写手”。
- 删除当前会话小标题。

替换为紧凑单行工具栏：

- 会话列表。
- 画风收藏。
- 记忆管理。
- 新建会话。
- 设置。
- 关闭。

这样给消息列表和输入栏留出更多纵向空间。

## 配置页同步规则

在提示词辅助配置页的 profile tab 中，点击“切换到此配置”时：

1. 调用现有 `switchProfile(i)` 切换当前 active profile。
2. 同步把 `_composeProfileIndex` 设置为 `i`。
3. 保存时通过现有 `setComposeProfile(i)` 持久化。

关键词抽取模型不跟随切换，避免把便宜小模型配置误改成主力编排模型。

## 历史画廊空态

当历史画廊不在加载，且 `_vm.history.isEmpty` 时，不再显示空网格。

显示：

- 空态图标。
- 文案：“请去创造更多美好回忆吧！”。
- 按钮：“去创作”。若 `onNavigate` 可用，点击切到生成页。

## 错误处理

- 学习记忆解析失败：不影响正常聊天，只忽略本次学习。
- 画风收藏图片压缩失败：提示用户保存失败，不写入半成品记录。
- 收藏图片文件丢失：卡片显示占位图，仍允许查看和编辑文本。
- 一键添加至对话栏失败：提示用户，并不删除收藏。

## 测试重点

1. 点击“切换到此配置”后，提示词编排模型随之改变，关键词抽取模型不变。
2. 手动新增记忆后，下一次输入命中 trigger 时，系统上下文包含该记忆。
3. 显式“记住/纠正”指令能保存记忆。
4. LLM 候选不会未经确认直接长期保存。
5. 画风收藏能从相册添加图片，图片被压缩并保存到本地。
6. 收藏详情能查看、编辑、删除，并一键添加到聊天输入栏。
7. 聊天抽屉顶部工具栏占用更少高度。
8. 历史为空时显示“请去创造更多美好回忆吧！”和“去创作”。
